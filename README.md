# Breast_Cancer_Perdiction_App
Build a machine-learning dashboard to predict whether a cell cluster is benign or malignant using Python and Streamlit

## Setup (Windows PowerShell)
- Create and activate a virtual environment:

```
py -3 -m venv .venv
./.venv/Scripts/Activate.ps1
```

- Install dependencies:

```
pip install -r requirements.txt
```

- Run the data loader test:

```
python ./Model/main.py
```

If you use VS Code, select the `.venv` interpreter for the workspace after creating it.
